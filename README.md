# LNbits Media

Assets, blog topics, and creative resources for the LNbits project.

## Structure

```
Articel_Blog/   Blog post drafts, images, and header graphics
Github_docs/    Screenshots and diagrams used in GitHub docs
```

## How to use this repo

**Writers** - Propose blog topics by opening an issue with the `blog-topic` label. Share drafts, get feedback, and collaborate.

**Designers** - Upload creative assets (logos, illustrations, social cards) via PR. Use the `design-asset` label on issues to request or offer graphics.

**Everyone** - Reference images from this repo in blog posts, docs, and social media. Use raw GitHub URLs for direct linking.

## Labels

| Label | Purpose |
|---|---|
| `blog-topic` | Blog post ideas open for discussion |
| `design-asset` | Logos, illustrations, social media graphics |
| `feedback` | Drafts or concepts requesting community input |
